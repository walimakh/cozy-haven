## Design Principles

### Visual Style
- **Aesthetic:** Clean, airy, minimalist with touches of warmth
- **Photography:** Bright, naturally lit, textural close-ups
- **Layout:** Generous whitespace = comfort and breathing room
- **Elements:** Soft rounded corners (8-12px radius)
- **Shadows:** Subtle elevation, not harsh
- **Grid:** 8px spacing system for consistency

### UI Design Patterns
- **Buttons:** Rounded corners, subtle hover effects
- **Cards:** Soft shadows, hover lift effects
- **Forms:** Clean inputs with focus states
- **Icons:** Simple, line-based, consistent stroke weight
- **Images:** High-quality, lifestyle-focused

### Accessibility
- Ensure sufficient color contrast (WCAG AA minimum)
- Readable font sizes (minimum 16px for body)
- Clear focus states for keyboard navigation
- Alt text for all images
- Semantic HTML structure

## Design Specifications

### Spacing System (8px Grid)
- 8px - Micro spacing (icon-to-text, tight elements)
- 16px - Small spacing (between related items)
- 24px - Medium spacing (between sections within cards)
- 40px - Large spacing (between major sections)
- 60px - Extra large spacing (page padding, hero spacing)

### Border Radius
- Buttons: 8px
- Cards: 12px
- Input fields: 6px
- Images: 8px

### Shadows
- Card hover: `0 4px 12px rgba(61, 57, 53, 0.08)`
- Elevated elements: `0 2px 8px rgba(61, 57, 53, 0.06)`
- Dropdown menus: `0 8px 24px rgba(61, 57, 53, 0.12)`

### Typography Scale
- H1: 48px / Line height 1.2
- H2: 36px / Line height 1.3
- H3: 28px / Line height 1.4
- H4: 22px / Line height 1.5
- Body: 16px / Line height 1.6
- Small: 14px / Line height 1.5
- Tiny: 12px / Line height 1.4

### Button Sizes
- Large: 48px height, 24px padding
- Medium: 40px height, 20px padding
- Small: 32px height, 16px padding

## Tips for Contest Success

### Stand Out Points
1. **Original concept** - Bedroom essentials is less saturated than fashion/electronics
2. **Strong brand identity** - Cohesive colors, typography, voice
3. **Attention to detail** - Proper spacing, alignment, consistency
4. **User-focused** - Clear navigation, helpful information, easy checkout
5. **Professional polish** - High-quality images (use Unsplash or similar), realistic content
6. **Thoughtful UX** - Size guides, material education, trust signals
7. **Responsiveness** - Both desktop and mobile are equally polished

### Common Pitfalls to Avoid
- Too much lorem ipsum (use realistic product descriptions)
- Inconsistent spacing and alignment
- Poor contrast/readability
- Cluttered layouts
- Missing interactive states
- Ignoring mobile experience
- Copied designs from existing brands
- Low-quality placeholder images

### Before Submission
- ‚úì Check alignment and spacing consistency
- ‚úì Verify all text is readable (contrast check)
- ‚úì Ensure consistent use of brand colors
- ‚úì Review for typos in all text
- ‚úì Test that mobile layouts make sense
- ‚úì Organize Figma file clearly
- ‚úì Add presenter notes to Figma frames
- ‚úì Export high-quality preview images
